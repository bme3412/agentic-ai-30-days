What This Demo Teaches

AutoGen takes a different approach: agents converse to solve problems. Rather than following predefined task sequences, they exchange messages dynamically, execute code, observe results, and iterate until they reach a solution.

  • Each agent's role and capabilities

Key Insights

  • Conversations are the unit of work—agents don't execute isolated tasks, they converse until the problem is solved
  • Code execution enables verification—agents can run code, see results, and iterate on failures
  • human_input_mode controls autonomy: ALWAYS (full oversight), TERMINATE (review at end), NEVER (autonomous)
  • GroupChatManager uses an LLM to dynamically select the next speaker based on conversation context
  • Use Docker for code execution—it provides isolation and prevents unsafe code from affecting your system
