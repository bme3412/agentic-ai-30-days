How Embeddings Capture Meaning

  • Semantic proximity: Similar meanings produce similar vectors
  • Fixed size: Any text length maps to the same vector dimensions
  • Learned features: Each dimension captures some aspect of meaning (not human-interpretable)
  • Model-specific: Different models produce incompatible embeddings

Understanding Cosine Similarity

  • Range: -1.0 (opposite) to 1.0 (identical)
  • Threshold: Typically >0.7 indicates strong similarity
  • Weaviate returns distance: distance = 1 - similarity (so 0 = identical)
  • Ignores magnitude: Only measures the angle between vectors

    • Keyword (BM25): Exact terms, code snippets, proper nouns, product codes
      • 
            
    • Semantic: Conceptual queries, "how to" questions, varied phrasing
      • 
            
    • Hybrid: General-purpose, RAG retrieval—best default for most cases
      • 
            
    • Alpha parameter: 0 = pure keyword, 1 = pure vector, 0.5 = balanced
