Key Concepts:

  • Host: The AI application that wants to use external tools
  • Client: Protocol handler within the host
  • Server: Service providing tools, resources, or prompts

Protocol Overview

The MCP protocol uses JSON-RPC 2.0 for communication. It follows a structured lifecycle:

  1. Initialize: Client sends capabilities, server responds with its capabilities
  2. Initialized: Client confirms, connection is ready
  3. Operation: Tools, resources, and prompts can be used
  4. Shutdown: Graceful disconnection
